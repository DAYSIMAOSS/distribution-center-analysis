# Distribution Center Analysis

This project analyzes the **efficiency, location, and geographical coverage** of distribution centers using **Python, Pandas, and data visualization techniques**.  
It provides insights to help optimize logistics operations and reduce transportation costs.

---

## Overview

The notebook performs:
- Data loading and cleaning for distribution centers and routes  
- Calculation of logistics metrics (distances, average time, coverage)  
- Visualization of clusters and spatial distribution  
- Identification of bottlenecks and optimization opportunities  

This project is ideal for studies in **logistics, supply chain management, and geospatial data analytics**.

---

## Technologies Used

- Python 3.x  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook
